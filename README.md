# FoodDeliveryApp

## Description
Приложение для заказа еды из ресторанов. Пользователь может просматривать список ресторанов, выбирать блюда из меню, добавлять их в корзину и оформлять заказ.

## Installation
1. Склонировать репозиторий: `git clone https://github.com/Nawruzbek/lab9.git`
2. Открыть проект в Xcode: `open FoodDeliveryApp/FoodDeliveryApp.xcodeproj`
3. Выбрать симулятор iPhone 15
4. Нажать `Run` (Cmd + R)

## Usage
- Главный экран — список доступных ресторанов
- Нажать на ресторан — открыть меню блюд
- Добавить блюда в корзину (кнопка "+")
- Перейти в корзину (иконка внизу)
- Оформить заказ (кнопка "Заказать")
